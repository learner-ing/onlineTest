# onlineTest
一个基于thinkphp5.1.39的在线考试系统

# 需求分析

### 1.用户分类：

主要分为三类，学生，教师和管理员。

#### 2.前台需求分析：

* 注册(学号，姓名，邮箱，密码，角色[老师和学生])。（前端验证）

* 登录(通过邮箱登录)。登录成功之后跳转到选择答题界面。（前端验证）
* 修改邮箱姓名。修改邮箱和密码（前端验证）
* 每一个答题有三种状态(开始时间[限时多久]，正在答题，答题已结束)。有标题和备注
* 点击对应的答题。即可获取到题目，题目题目分为单选和多选。底部显示答题倒计时(分钟为单位)，到时间自动提交。
* 查看每一次答题的排名并显示本次答题的分数

#### 3.后台需求分析：

* 主要分为三个模块（用户管理[教师和学生]，题目管理，答题管理）
* 用户管理：分为学生管理和教师管理[删除，重置密码，查看该用户每一次答题的排名]。管理员的添加和删除
* 题目管理：添加题目[单选和多选]。删除题目。
* 答题管理：添加答题，删除答题，手动结束。
* 答题详情：设置单选和多选分数，设置单选和多选题目数量[题目随机从数据库选择]。
* 查看答题：查看本次答题的用户[老师和学生]并排名。
